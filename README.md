# PRIME TECH – AI-Driven 3D Portfolio Landing Page

## Overview

This project was developed as part of the Prime Tech AI-Expert Web Intern Assignment.

The goal was to create a modern, AI-inspired, single-page landing page that demonstrates:

- Interactive 3D graphics
- Smooth animations
- AI-generated content
- Modern UI/UX design
- Responsive development
- Fast performance

## AI Workflow & Prompt Engineering

### Content Generation

AI-assisted copywriting was used to generate:

- Headlines
- Service descriptions
- Marketing content
- Workflow explanations

Example Prompt:

"Generate modern SaaS landing page copy for an AI technology company focused on innovation, automation, and growth."

---

### 3D Asset Research

The assignment suggested:

- Spline AI
- Meshy AI
- Luma AI
- Vectary AI

These tools were evaluated during planning.

To ensure performance, responsiveness, and deployment simplicity, the final implementation used an optimized Three.js-based interactive 3D object.
